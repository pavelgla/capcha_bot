📖 Инструкция по использованию бота

Шаг 1 — Укажите Telegram ID

Введите ваш Telegram ID в поле выше. Без него бот не сможет определить, какому аккаунту принадлежит чат.

Как узнать ID: напишите /start боту @userinfobot — он пришлёт ваш ID.

Шаг 2 — Добавьте бота в чат

  1. Откройте ваш Telegram-чат (должна быть супергруппа).
  2. Добавьте @bot_captha_bot как администратора.
  3. Выдайте боту следующие права администратора:
    • ✅ Ограничивать участников
    • ✅ Удалять сообщения
Важно: Капча работает только в супергруппах. Обычная группа автоматически конвертируется в супергруппу при первом добавлении администратора, либо при достижении 200 участников. Если бот не реагирует на вступление — убедитесь, что чат является супергруппой.

Шаг 3 — Привяжите чат командой /setup

Напишите в чате (где уже есть бот-администратор):

/setup

Бот найдёт ваш аккаунт по Telegram ID и привяжет чат. После этого чат появится в дашборде.

Команды бота

Команда Где работает Описание
/setup В чате Привязывает чат к вашему аккаунту. Выполняется один раз.
/unmute @username В чате Вручную снимает ограничения с пользователя. Доступно администраторам чата.
/mutestat В чате Показывает статистику капчи за всё время (вступило / прошли / провалили / таймаут).
/banned В чате Список пользователей, замьюченных навсегда (не прошли капчу все попытки).

Как работает капча

  1. Новый участник вступает в чат — бот сразу ограничивает его (не может писать).
  2. Бот отправляет приветственное сообщение с капчей в виде арифметического примера (например, «3 + 7 = ?») и несколькими кнопками-вариантами.
  3. У участника есть N попыток и X секунд на ответ (настраивается в панели).
  4. Прошёл капчу — ограничения снимаются, участник может писать.
    Не прошёл (исчерпал попытки) — замьючен навсегда, попадает в список /banned.
    Таймаут — участник остался мьютнутым; может попробовать снова при следующем вступлении или размьютить через /unmute.

Настройки чата

Для каждого привязанного чата доступны настройки через дашборд → кнопка «Настроить»:

ПараметрПо умолчаниюОписание
Таймаут капчи 300 сек (5 мин) Время, за которое нужно ответить на капчу. Минимум 30 секунд.
Количество попыток 2 Сколько раз можно ошибиться, прежде чем получить вечный мьют.
Капча включена вкл Можно временно отключить проверку (например, при массовом добавлении участников).
Приветственное сообщение стандартное Текст, который бот отправляет новому участнику вместе с капчей. Поддерживается произвольный текст.
